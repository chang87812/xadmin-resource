# xadmin系列——资源

### 前后分离的通用后台管理系统，前端基于Ant Design Pro-Vue，后端基于SpringBoo 2.x。

##### 包含：定时任务管控、基于JWT的配合Shiro的鉴权、一键生成前后端增删改查代码、基于RBAC权限管控、用户管理、组织架构、各类日志(操作日志/定时任务日志/登录日志)等模块
